Dictyostelid de novo assemblies

Dictyostelid Quickload Server

The de novo transcriptome assembly data are made available here as an IGB Quickload server.

Summary

This landing page describes how to access data associated with the "de novo transcriptome assembly of four dictyostelid species" study published in BMC Genomics doi:TBA

The data have been made freely available as a Quickload server for easy browsing within Integrated Genome Browser (IGB). See below for instructions in how to set it up.

Setting up

Firstly, download and install Integrated Genome Browser (IGB) locally

Then, set up a new data server in the Data Access panel as described in the manual.

InfoSet the url as 'https://compbio.lifesci.dundee.ac.uk/Quickload/Dictyostelid_assemblies/'

Now, in the main window, choose one of the four species: Dictyostelium discoideum, Dictyostelium fasciculatum, Dictyostelium lacteum or Polysphondylium pallidum and our data should appear ready to use in the Data Access panel.
